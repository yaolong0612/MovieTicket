## Quick Start

### Application Test

**On Mac or Linux**:

```sh
./mvnw test

**On Windows**:

```

### Application Package

**On Mac or Linux**:

```sh
./mvnw package
```

**On Windows**:

```sh
mvnw.cmd package
```

### Application Run

**On Mac or Linux**:

```sh
./mvnw spring-boot:run
```

**On Windows**:

```sh
mvnw.cmd spring-boot:run
```
